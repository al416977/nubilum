NUBILUM

Sube, reduce, limpia y prepara nubes de puntos 3D de forma sencilla. La plataforma está pensada para visualizar archivos pesados, generar versiones más ligeras y dejar la nube lista para futuras tareas como clasificación de superficies o división por zonas.

Subir archivo nuevo Abrir proyectos

¿Qué puede hacer la plataforma?

  • Reducir una nube de puntos para que cargue más rápido.
  • Eliminar parte del ruido o puntos erróneos.
  • Visualizar una versión más ligera en 3D.
  • Preparar la nube para clasificar superficies como suelo, pared o techo.
  • Guardar el trabajo por proyectos para revisarlo más tarde.

Formatos admitidos

  • .xyz
  • .pts
  • .ply
  • .pcd
  • .las y .laz

Si el archivo incluye color RGB, la plataforma intentará conservarlo en la visualización. Para una experiencia más fluida, se recomienda trabajar con nubes reducidas o ya procesadas cuando el archivo sea muy grande.

Cómo funciona

El flujo recomendado es muy simple:

1. Subir archivo 2. Ajustar parámetros 3. Procesar 4. Abrir proyectos 5. Clasificar y visualizar

La carga y el preprocesado se realizan desde la página de subida. La visualización, clasificación por clases y descarga de resultados se gestionan desde la página de proyectos.
